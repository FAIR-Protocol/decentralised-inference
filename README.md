### File Structure
├── poc-ng-wireframes
├── client
├── .gitignore
├── LICENSE
└── README.md

## Installation
Initialize Submodule:
```
git submodule update  --init
```

Enter Client directory and install dependencies:
```
cd client
npm install
```

## Running
Initiate the application in a local environment:
```
npm run dev
```
